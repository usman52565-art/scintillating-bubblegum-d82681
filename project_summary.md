## 🎨 Customization Examples

### Change Brand Color
In `assets/css/style.css`:
```css
--primary: #6366f1; /* Change to your color */
```

### Add New Course
In `assets/js/script.js`, add to courses array:
```javascript
{
  id: 7,
  title_en: "New Course",
  title_pt: "Novo Curso",
  price: "$49.99",
  hotmart_link: "your-link",
  // ... more fields
}
```

### Update Company Info
In HTML files, search for "Smart Online Academy" and replace with your name.
