## 🔧 **CUSTOMIZATION GUIDE**

### Change Brand Color
File: `assets/css/style.css`
```css
--primary: #6366f1; /* Change to your color */
```

### Add New Course
File: `assets/js/script.js`
```javascript
{
  id: 7,
  title_en: "Your Course",
  price: "$49.99",
  hotmart_link: "your-link",
  // ... more fields
}
```

### Update Company Info
Search & replace in all HTML files:
- "Smart Online Academy" → Your company name
- Contact email, phone, WhatsApp
- Hotmart product links

See `COURSE_MANAGEMENT.md` for detailed instructions.
